Principais formatações

Negríto / Destaque

Nessa frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, tenho um termo em destaque/ usando a tag strong que ela é (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semantica)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrafo, temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Texto subscrito

Para inserir coisas do tipo H2O

Descrições dos códigos

Para se fazer um texto em negrito é necessário usar a tag b

Para deixar uma frase em destaque é necessário usar a tag strong

Para usar italico é necessário envelopar com CRTL,SHIFT,P,I

Para dar enfase a uma frase é necessário envelopar com CTRL,SHIFT,P,ENTER,EM

Para marcar um texto é necessario envelopar com CTRL,SHIFT,P,ENVELOPAR COM ABREVIATURA,MARK

Para diminuir uma frase é necessario envelopar com CTRL,SHIFT,P,ENVELOPAR COM ABREVIATURA,SMALL

Para marcar um texto como excluido é necessário utilizar a tag ins
